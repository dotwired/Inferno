# Function Reference

## inferno_preview()

### Description

Wrapper function for [`Inferno_Preview()`][class_inferno_preview] class.

### Usage

```php
<?php
$args = array(
  'src' => false,
  'width' => false,
  'height' => false,
  'permalink' => false,
  'crop' => true,
  'effect' => 'default',
  'module' => null
);

echo inferno_preview($args);
?>
```

### Parameters

See the [`Inferno_Preview()`][class_inferno_preview] for a description of the arguments passed to `inferno_preview()`
